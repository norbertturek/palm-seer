# PalmSeer

Aplikacja do analizy chiromancji dłoni wykorzystująca sztuczną inteligencję.

## Technologie

- React
- Vite
- Tailwind CSS
- Shadcn UI
- Supabase

## Konfiguracja i Uruchomienie

### Wymagania wstępne

- Node.js
- npm / pnpm / bun

### Instalacja

1. Sklonuj repozytorium:
   \`\`\`bash
   git clone git@github.com:norbertturek/palm-seer.git
   \`\`\`

2. Zainstaluj zależności:

\`\`\`bash
npm install
\`\`\`

### Uruchomienie lokalne

Uruchom serwer developerski:

\`\`\`bash
npm run dev
\`\`\`

Aplikacja będzie dostępna pod adresem \`http://localhost:8080\`.

## Zmienne środowiskowe

Upewnij się, że skonfigurowałeś odpowiednie zmienne środowiskowe dla Supabase i API AI.
